# speedss
### 自动生成一个speedss的试用账号
生成一个高质量的试用3天的SS 账户，提供梯子，遨游世界

技术：
1. JSOUP:解析目标网页的源代码，把所需要的网页的内容获取
2. OKGO 请求接口或者网页的请求框架
3. easypermissions 动态申请权限
4. 利用系统自带的系统下载器下载APP，并且适配android 8

### Version 1
自动随机生成一个账户，实现该账户的注册
### Version 1.1
生成一个固定服务器的随机账户的邮件账户，突破邮件验证码的限制
### Version 2
下载android 客户端
